# 👗 StyleAI

An AI-powered outfit recommendation platform built using Next.js, FastAPI, and Google Gemini AI.

## Features

- 🤖 AI Outfit Recommendations
- 👕 Personalized Style Suggestions
- 📸 Image-Based Fashion Analysis
- 🎨 Color Matching
- 💬 AI Fashion Assistant
- 📱 Responsive Design

## Tech Stack

- Next.js
- React
- TypeScript
- FastAPI
- Google Gemini AI
- Tailwind CSS
